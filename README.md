# movies_stock

## How to run
1. Clone this repository.
2. Create `.env` file and copy-paste the content of `envsample` to `.env`.
3. `npm install`.
4. `npm run start`.
5. Open (http://localhost:3000/).

## How to run unit testing
- `npm run test`.

##### Extra Notes:
- Node v12.13.1 and NPM v6.12.1 are used to build this project.
